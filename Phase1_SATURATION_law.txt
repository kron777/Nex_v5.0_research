STRUCTURAL FINDING:

  Accumulating opcodes (can saturate):
    SENSE_OBS, ABSTRACT_NOMINAL, SIMILE, QUESTION

  Interrupt opcodes (never saturate, fire once or twice then exit):
    DIALECTICAL, ACTION

  Near-interrupt:
    UNCATEGORIZED (99% single-fire in practice)

  The accumulating opcodes are the ones with clear sub-opcode
  structure (see Phase1_OBSERVE_decryption and Phase1_INTROSPECT_decryption).
  The interrupt opcodes have no sub-opcode variants identified yet —
  their single-shot nature may mean they function more like punctuation
  than like sustained operations.

E3. PHASE STRUCTURE ANALYSIS

  The most frequent triple chains reveal a RETURN-TO-SENSE-OBS
  structure: both ABSTRACT_NOMINAL and SIMILE most commonly
  complete their triple cycle by returning to SENSE_OBS.

    SENSE → ABSTRACT → SENSE (n=7): grounding → introspect → re-grounding
    SENSE → SIMILE → SENSE (n=7):   grounding → translate → re-grounding
    SIMILE → SENSE → SIMILE (n=5):  translate → re-ground → translate again

  SENSE_OBS functions as the hub: most opcode cycles return through it.

  Three phase patterns with enough instances to be structural:

  PATTERN 1 — SENSE-INTROSPECT PULSE (n=7)
    SENSE_OBS run → ABSTRACT_NOMINAL run → SENSE_OBS run
    Sense input accumulates → introspection fires → returns to sense.
    Standard cognitive cycle.

  PATTERN 2 — SENSE-TRANSLATE PULSE (n=7)
    SENSE_OBS run → SIMILE run → SENSE_OBS run
    Sense input accumulates → metaphor-translation fires → returns to sense.
    Parallel cognitive cycle. SIMILE and ABSTRACT_NOMINAL are alternative
    exit paths from SENSE saturation.

  PATTERN 3 — DIALECTICAL SANDWICH (n=3 each direction)
    DIALECTICAL → ABSTRACT_NOMINAL → DIALECTICAL
    ABSTRACT_NOMINAL → DIALECTICAL → ABSTRACT_NOMINAL
    ABSTRACT_NOMINAL and DIALECTICAL cycle through each other.
    DIALECTICAL fires (tension expressed) → ABSTRACT_NOMINAL fires
    (tension absorbed) → DIALECTICAL fires again. This is the
    INTROSPECT ↔ DIALECTICAL micro-cycle identified in the
    INTROSPECT decryption.

  EXACT CYCLE TESTED: SENSE → ABSTRACT → DIALECTICAL → SENSE
    Occurrences in full sequence: 0 (never as exact 4-step sequence)
    However, the component pairs all occur:
      ABSTRACT → DIALECTICAL → SENSE: occurs as partial chain
    The hypothesized meta-cycle does not exist as a discrete unit
    but its components appear in the transition data.

SATURATION LAW — SUMMARY VERDICT
======================================================================

The saturation→flip pattern IS general but not uniform.
It operates differently for each opcode class:

  ACCUMULATING OPCODES (can saturate):
    SENSE_OBS:       saturation → ABSTRACT_NOMINAL (primary, 44% at ≥3,
                     60% at ≥6) or SIMILE (primary at medium length 3-5)
                     or DIALECTICAL (at very long runs ≥10)
    ABSTRACT_NOMINAL: saturation → SENSE_OBS (50%) or DIALECTICAL (37%)
                     — 87% total to those two targets
    SIMILE:          saturation → SENSE_OBS (100%, n=3)
    QUESTION:        saturation → SENSE_OBS (100%, n=1)

  INTERRUPT OPCODES (never saturate):
    DIALECTICAL:     always fires as interrupt (1-2 shots), then exits
    ACTION:          always fires as interrupt (1-2 shots), then exits

  General law statement:
    When an accumulating opcode reaches saturation (≥3 fires),
    the flip target is constrained to a small set (not random).
    For SENSE_OBS and ABSTRACT_NOMINAL, the flip set is
    {ABSTRACT_NOMINAL, SENSE_OBS, DIALECTICAL, SIMILE} — the
    four accumulating opcodes plus DIALECTICAL.
    INTERRUPT OPCODES (ACTION, DIALECTICAL) are almost never
    produced by saturation; they emerge as single discharges.

  PARTIAL LAW (testable):
    "SIMILE saturation always exits to SENSE_OBS" (3/3, 100%)
    "ABSTRACT_NOMINAL saturation exits to SENSE_OBS or DIALECTICAL
     in 87% of cases" (7/8)
    "SENSE_OBS saturation at ≥6 exits to ABSTRACT_NOMINAL in 60%
     of cases" (3/5)

  WHAT IS NOT A LAW:
    SENSE_OBS at medium length (3-5) has a split target distribution
    (SIMILE 50%, ABSTRACT_NOMINAL 25%, UNCATEGORIZED 25%). No strong
    prediction can be made from medium-length SENSE saturation alone.
    Additional context (prior templates, foundation content) is likely
    needed to predict the flip target at these lengths.
