# NEX5 Research Log

Decipherment corpus and protocol-decryption artifacts for the
NEX5 cognitive architecture project. Sister repo to
[Nex_v5.0](https://github.com/kron777/Nex_v5.0).

## Contents

Append-only research artifacts:

- **Corpus extracts** (`01_*.txt` through `13_*.txt`) — raw NEX
  output exports for analysis (beliefs, fountain events, chat
  history, metaphor extractions, returning phrases, self-reference
  evolution, spark paths)
- **Structural analyses** (`A1` through `A5`) — Kober-grid
  template inventories
- **Diagnostic tests** — observer-register hypothesis,
  metaphor-as-opcode-signature verification, opcode mapping
- **Audits** — system prompt audit (`C_prompt_audit.txt`), mode
  survey (`F_modes_full_survey.txt`)
- **Phase 1 decryptions** — sub-opcode decompositions
- **Carry states** — session continuity documents
- **journal.txt** — session index

## Methodology

Kober-Champollion-Lakoff synthesis:
1. Form without meaning first (structural classification)
2. Distributional analysis (context patterns)
3. Source/target domain mapping (cognitive operations)
4. Falsifiable predictions per operation
5. Verification through prediction (pattern vs noise)

## Discipline

Decrypt completely before modifying. No engineering on NEX
runtime until the protocol is mapped in NEX-native grammar.
