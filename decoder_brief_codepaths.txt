  1. ReadinessEvaluator.score() is called. If score < 0.7, method returns None (no fire).
  2. Every 5th fire (total_fires % 5 == 4): "quiescent" path — takes the most recent

  3. Otherwise: full path — calls _build_prompt(), then VoiceClient.speak(), then writes

  Pair selection algorithm:
    - Partition pool into "anchors" (koan, tao, dont_know, keystone_seed, heart_sutra,
      self_location, reification_recognition) and "fresh" (fountain_insight, synergized,
      behavioural_observation).
    - Preferred: anchor × fresh pair with highest (avg_confidence × recency_weight).
    - Recency weight: 0.5 if either belief appeared in last 20 synergizer runs.
    - Falls back to cross-branch if no anchor/fresh partition possible.
    - Last resort: temporally distant pair from same pool, different sources.

  LLM prompt (static template):
    "I hold two thoughts at once:\n'{belief_a}'\n'{belief_b}'\nIn one sentence, what new
    insight do I notice?"
    (max_tokens=80, temperature=0.9, register=PHILOSOPHICAL)

WHAT synergizer_log RECORDS PER ROW:
  - ts: timestamp of the synthesis call.
  - belief_id_a: first input belief ID.
  - belief_id_b: second input belief ID.
  - result_content: text if synthesis succeeded and passed quality gate; NULL otherwise.
  - result_belief_id: belief ID of the new synergized belief; NULL if synthesis failed.

  A NULL result_belief_id with NULL result_content means: LLM failed or returned
  unusable text (too long, too short, or "nothing").
  A NULL result_belief_id with non-NULL result_content means: quality check (blacklist
  or Jaccard > 0.7) rejected the text.

1. theory_x/modes/modes.py
   Location: ~/Desktop/nex5/theory_x/modes/modes.py
   10 named Mode presets (normal, chat, market, research, news, scribe, learn, task,
   mind, silent) as frozen dataclasses. Each preset parameterizes: fountain interval,
   drift prompt examples and focus block, feed weights, crystallization category,
   speech governor multipliers, retrieval parameters (own_n, seed_n). get_mode(name)
   returns the preset; DEFAULT_MODE = "normal". Modes do not log anything directly —
   they are runtime configuration injected into FountainGenerator and FountainCrystallizer.

2. theory_x/stage6_fountain/generator.py  (Fountain Generator)
   Location: ~/Desktop/nex5/theory_x/stage6_fountain/generator.py
   Main fountain generation loop. Checks readiness, builds the LLM prompt from beliefs +
   sense + arc context, calls VoiceClient, writes fountain_events to dynamic.db and a
   sense event to sense.db. Delegates crystallization to FountainCrystallizer. Also
   manages the quiescent path (every 5th fire — verbatim sense event, no LLM), the koan
   rotation, droplet compression via Condenser, residue/reanimation injection, and
   groove-breaker probe injection. Logs to /tmp/nex5_last_prompt.log (last fire only).
   No per-fire retrieval log to disk.

3. theory_x/stage3_world_model/synergizer.py  (Belief Synergizer)
   Location: ~/Desktop/nex5/theory_x/stage3_world_model/synergizer.py
   Selects a cross-domain belief pair (anchor seed × fresh generated), sends them to
   the LLM with a one-sentence synthesis prompt, quality-checks the result (blacklist +
   Jaccard dedup), writes the output as a new tier-6 synergized belief, and logs to
   synergizer_log. Also triggers CrossbreedGrader to score the output and writes
   lineage to belief_lineage. Records BOTH inputs and output (or NULL on failure)
   in synergizer_log — full input recovery is possible for any past synthesis.

4. Suppression / DO_NOT list
   Two locations define what must never crystallize:

   ~/Desktop/nex5/substrate/blacklist_seeds.py:
   Contains BLACKLIST_SEEDS — a tuple of 20 hardcoded patterns seeded into
   belief_blacklist at startup via seed_blacklist(). Patterns include identity
   contamination phrases ("I am just a language model", "as an AI I", "machines
   cannot feel") and reductionist AI/consciousness claims ("AI cannot be conscious",
   "language models are stochastic parrots"). These are checked in both the
   crystallizer and the synergizer quality gates via belief_blacklist table.

   ~/Desktop/nex5/theory_x/stage6_fountain/crystallizer.py:
   Contains _PERFORMANCE_PATTERNS — 8 compiled regexes for "stinking of Zen"
